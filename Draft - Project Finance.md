# Project Finance

- Relates to the financial framework within which the project operates.
- Covers the budget available for the project, how it has been approved and allocated, and the financial boundaries that shape decisions around scope and resourcing. Also covers procurement constraints, cost tracking, invoicing arrangements, ongoing running costs, licensing, and any financial risks or dependencies that could affect delivery.
- Related subjects are Project Governance (Business & Operations), which covers the decision-making structures within which financial authority sits.
