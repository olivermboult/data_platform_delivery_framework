# Data Sources & Ingestion

- Relates to the origin points of all data that will flow through the solution.
- Covers the identification and documentation of each source system, including its type, location, size, refresh frequency, and current status. Also covers ingestion methods, access and connectivity requirements, data sharing agreements or governance approvals needed before access can be granted, and any third-party ownership arrangements that affect how source systems can be used within the solution.
- Related subjects are Data Architecture (Data & AI), which covers how the characteristics of source systems influence the structural design of the solution.
