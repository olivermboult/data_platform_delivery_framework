# Data Ownership

- Relates to the accountability structures that govern data within the platform after it has been ingested and transformed. 
- Covers who is responsible for the accuracy, quality, and appropriate use of data within each domain and how that responsibility is maintained over time.
- Related subjects are Roles & Access (Platform & Security), which covers how data ownership is enforced through platform permissions, Data Quality (Data & AI), which covers the quality standards data owners are responsible for upholding, and Data Architecture (Data & AI), which covers how domain structures within the platform reflect ownership boundaries.
